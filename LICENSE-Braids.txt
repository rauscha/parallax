Attribution for vendored DSP code
==================================

The directories `dsp/vendor/braids/` and `dsp/vendor/stmlib/` contain source
code from Émilie Gillet's Mutable Instruments firmware, vendored from:

  https://github.com/pichenettes/eurorack

Specifically:
  - braids/ — the Braids macro-oscillator firmware
                  (eurorack/braids/, MIT)
  - stmlib/ — shared DSP utilities used by Braids
                  (github.com/pichenettes/stmlib, MIT — vendored as a git
                  submodule of eurorack; we ship only the DSP-relevant subset)

All retained files preserve their original Émilie Gillet copyright headers and
the MIT license text below applies to every file under `dsp/vendor/`.
